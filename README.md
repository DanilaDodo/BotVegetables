# VK Vegetable Ordering Bot

Бот для оформления заказов овощей между пиццериями и последующего согласования заказа администратором.

Проект создан для автоматизации оформления заказов между пиццериями сети.

## Возможности

- выбор пиццерии через inline-кнопки;
- пагинация списка пиццерий;
- формирование корзины;
- выбор товаров через callback-кнопки;
- ввод количества товара сообщением;
- хранение состояния пользователя через FSM vkbottle;
- согласование заказа администратором (в разработке).

## Стек

- Python 3.x
- vkbottle
- JSON (временное хранилище данных)

## Архитектура

Бот построен на событийной модели vkbottle.

Состояние пользователя хранится в FSM:
- выбранная пиццерия;
- текущий товар;
- корзина заказа.

Интерфейс реализован через callback-кнопки и inline-клавиатуры.

## Запуск

1. Установить зависимости:

pip install -r requirements.txt

2. Создать VK-группу и получить токен сообщества.

3. Вставить токен в main.py.

4. Запустить:

python main.py
